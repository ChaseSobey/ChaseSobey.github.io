Embrasure Secrets

Embrasure is an open-source, self-hosted secrets management tool built on Amazon Web Services (AWS) for small teams seeking simplicity and security. It protects your application secrets through a combination of at-rest and in-transit encryption, access control, injection-at-runtime, and a custom built secure API. Built using Node.js and Amazon Web Services, it is designed for easy installation, set up and integration with your AWS applications.

For the developer, Embrasure provides a simple to use CLI interface to perform all secrets and team permission management. Furthermore, Embrasure injects secrets at runtime and redacts secrets from getting logged on stdOut and StdErr.
